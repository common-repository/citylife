== Description ==

The easiest way to include CityLife in your website. Create some custom discover feeds with the channels you want.

== Installation ==

1. Once activated, click the "WP CityLife" link under your settings tabs.
2. Edit the settings to your preference.
3. Test it out in a post or page.


== Frequently Asked Questions ==

= I've installed the plugin, now what? =

Once installed and activated, a link should appear in your left navigation panel within your WP-ADMIN section. Click on the "WP CityLife" link and follow the on screen instructions.
